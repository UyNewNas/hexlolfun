# 📌 附录：池子机制说明

## 动态智能池机制

海克斯大乱斗的池子是**动态智能池**，会根据英雄特性自动过滤不相关的海克斯：

1. **技能类型过滤**
   - 无AP加成的英雄 → 不会刷出法强相关海克斯（如超强大脑、任务巫师帽）
   - 无AD加成的英雄 → 不会刷出AD相关海克斯

2. **定位过滤**
   - 无位移技能 → 不会刷出位移类海克斯（如全凭身法、暗影疾奔）
   - 无硬控（定身）技能 → 不会刷出需要硬控触发的海克斯（如残忍、坚如磐石）

3. **英雄池子纯净度示例**
   - **蒙多**：技能全与生命值挂钩，无位移、无硬控、无法强 → 池子极净，容易拿到坦克类海克斯（坦克引擎55%登场率、钢化你心36.9%）
   - **瑞兹**：池子也非常干净，核心海克斯"由心及物"和"物转法"登场率均超30%
   - **火男**：容易拿到天胡海克斯魔法飞弹和炼狱导管
   - **阿狸**：池子极杂，AD类、AP类、辅助类海克斯混杂，容易拿到废海克斯
