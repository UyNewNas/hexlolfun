# 英雄联盟海克斯大乱斗3.0 海克斯池子

> 版本：16.12 | 上线时间：2026年6月11日 | 热补丁：2026年6月13日

---

## 一、版本核心变动

- **移除** 羁绊系统，部分羁绊效果（叠角龙、下雪天等）转为独立符文
- **删除** 39个旧有强化符文
- **新增** 62个强化符文（11白银 + 28黄金 + 23棱彩）
- **实装** 技能强化符文机制 — 仅适用于英雄特定技能，产生差异化效果
- **通行证升级** 2.0版本，完成3场对局可领限定皮肤（活动至2026年7月15日）

### 调整的既有符文（3个）

| 符文 | 调整内容 |
|------|---------|
| **亮出你的剑** | 根据攻击距离（500-650码）线性提升属性加成，最高达100%，远程高射程英雄显著受益 |
| **利刃华尔兹** | 修正落点逻辑，确保最终位置锁定初始目标 |
| **转得我眩晕了** | 攻击距离低于400码的英雄提升至400码，优化近战使用体验 |
