## 1. 伤害类型

英雄联盟有三种伤害类型：

| 类型 | 被谁减免 | 穿透方式 | 典型来源 |
|------|---------|---------|---------|
| **物理伤害** | 护甲 | 穿甲(固定)、百分比护甲穿透 | 普攻、AD技能 |
| **魔法伤害** | 魔法抗性 | 固定法穿、百分比法穿 | AP技能 |
| **真实伤害** | 无法被减免 | 无需穿透 | 薇恩W、盖伦R、征服者 |

---

## 2. 原始伤害计算

### 2.1 技能原始伤害

```
原始伤害 = 技能基础伤害 + (AD × AD加成率) + (AP × AP加成率)
```

- `技能基础伤害`: 随技能等级提升
- `AD加成率`: 技能描述的 AD 系数 (如 1.2 = 120% AD)
- `AP加成率`: 技能描述的 AP 系数 (如 0.8 = 80% AP)

### 2.2 普攻原始伤害

```
普攻伤害 = 面板攻击力 (AD)
```

## 3. 抗性减伤公式

护甲和魔抗使用相同的计算公式。

### 3.1 伤害减免率

```
物理减免率 = 护甲 / (100 + 护甲)

魔法减免率 = 魔抗 / (100 + 魔抗)
```

> 示例: 100 护甲 → 100/(100+100) = **50%** 物理减伤

### 3.2 伤害乘数

```
伤害乘数 = 100 / (100 + 抗性)
```

> 示例: 100 护甲 → 100/(100+100) = **0.5** (受到50%伤害)

### 3.3 抗性收益特点

- **每点抗性增加 1% 有效生命值** (线性收益，无边际递减)
- 永远无法达到 100% 减伤
- 减伤率曲线: 前期提升快，后期提升慢 (但 EHP 是线性增长)

## 4. 穿透机制

### 4.1 穿透结算顺序

```
实际抗性 = (总抗性 × (1 - 百分比穿透)) - 固定穿透
```

**结算优先级:**
1. 固定减抗 (如黑切、皇子Q)
2. 百分比减抗 (如黑切叠满)
3. 百分比穿透 (如多米尼克领主的致意 35%、虚空之杖 40%)
4. 固定穿透 / 穿甲 (如锯齿短匕、幽梦之灵)

### 4.2 百分比穿透

```
百分比穿透后的抗性 = 总抗性 × (1 - 穿透百分比)
```

常见百分比穿透:
- 多米尼克领主的致意: **35%** 护甲穿透
- 虚空之杖: **40%** 法术穿透
- 最后的轻语: **20%** 护甲穿透

### 4.3 固定穿透与穿甲的区别

**固定法术穿透** 直接作用于魔抗:

```
实际魔抗 = 魔抗 - 固定法穿
```

**穿甲 (Lethality)** 需要等级换算后才作用于护甲:

```
实际护甲穿透 = 穿甲 × (0.6 + 0.4 × 英雄等级 / 18)
```

> 1级时: 1穿甲 ≈ 0.267 护甲穿透
> 18级时: 1穿甲 = 1 护甲穿透

## 5. 最终伤害结算流程

### 完整结算步骤

```
Step 1: 原始伤害 = 技能基础伤害 + (AD × 加成率) + (AP × 加成率)

Step 2: 真实伤害 → 直接输出，跳过后续步骤
         物理伤害 → 使用护甲计算
         魔法伤害 → 使用魔抗计算

Step 3: 实际抗性 = (目标总抗性 × (1 - 百分比穿透)) - 固定穿透

Step 4: 最终伤害 = 原始伤害 × 100 / (100 + 实际抗性)
```

### 公式汇总

```
物理: 最终伤害 = AD × 100 / (100 + 实际护甲)
魔法: 最终伤害 = AP伤害 × 100 / (100 + 实际魔抗)
真实: 最终伤害 = 原始伤害值 (不受任何减免)
```

## 6. 有效生命值 (EHP)

### 6.1 计算公式

```
对物理伤害有效生命 = 最大生命值 × (100 + 护甲) / 100
对魔法伤害有效生命 = 最大生命值 × (100 + 魔抗) / 100
```

### 6.2 关键结论

- **每 1 点护甲 = +1% 对物理伤害的有效生命**
- **每 1 点魔抗 = +1% 对魔法伤害的有效生命**
- 抗性不存在"边际递减"——EHP 增长是严格线性的
- 实际感受到的"递减"只是减伤率曲线的视觉错觉

### 6.3 示例

| 护甲 | 减伤率 | 有效生命(对物理) |
|------|--------|-----------------|
| 0 | 0% | 100% HP |
| 50 | 33.3% | 150% HP |
| 100 | 50% | 200% HP |
| 200 | 66.7% | 300% HP |
| 300 | 75% | 400% HP |

> 1000HP、100护甲的英雄, 等效于 **2000HP** 对物理伤害的承伤能力

## 7. 暴击伤害

### 7.1 基础暴击

```
默认暴击伤害 = 175% AD (S16赛季调整)
```

### 7.2 平均伤害倍率

```
平均伤害倍率 = 1 + 暴击率 × (暴击伤害倍率 - 1)
```

### 7.3 带无尽之刃

```
无尽之刃: 暴击伤害 = 225% AD
```

## 9. 负抗性伤害加成

当穿透后目标抗性为负数时，伤害会进一步提升：

```
负抗性伤害乘数 = 2 - 100 / (100 - 实际抗性)
```

| 实际抗性 | 伤害乘数 | 伤害加成 |
|----------|---------|---------|
| 0 | 1.00x | 0% |
| -10 | 1.09x | +9% |
| -25 | 1.20x | +20% |
| -50 | 1.33x | +33% |
| -100 | 1.50x | +50% |

## 11. 实战计算示例

### 示例 1: 刺客 vs 脆皮

```
劫 (18级, 300AD, 50穿甲, 30%护甲穿透)
→ 目标: 凯特琳 (100护甲)
→ 技能原始伤害: 600 (物理)

实际护甲 = max(0, 100 × (1-0.30) - 50) = 20
伤害乘数 = 100 / (100 + 20) = 0.833
最终伤害 = 600 × 0.833 = 500
```

### 示例 2: 法师 vs 坦克

```
拉克丝 (400AP, 虚空之杖40%法穿)
→ 目标: 蒙多 (200魔抗)
→ R技能原始伤害: 1000 (魔法)

实际魔抗 = 200 × (1-0.40) = 120
伤害乘数 = 100 / (100 + 120) = 0.455
最终伤害 = 1000 × 0.455 = 455
```

### 示例 3: 真伤技能

```
薇恩 W技能 (3环, 12%最大生命值真实伤害)
→ 目标: 3000HP的坦克

最终伤害 = 3000 × 12% = 360 (无视任何抗性)
```

### 示例 4: 穿透顺序完整示例

```
攻击者: 攻击力200, 穿甲30, 百分比穿透35%
目标: 护甲150

Step 1 - 百分比穿透: 150 × (1-0.35) = 97.5
Step 2 - 固定穿透(18级): 97.5 - 30 = 67.5
Step 3 - 伤害: 200 × 100/(100+67.5) = 200 × 0.597 = 119.4

如果不带穿透: 200 × 100/(100+150) = 80
穿透提供了 39.4 点额外伤害 (+49%)
```

## 12. 减伤/增伤 Modifiers

除了抗性之外，还有其他独立乘区的减伤/增伤效果：

### 12.1 独立减伤

| 来源 | 效果 |
|------|------|
| 兰德里的折磨 | 造成伤害时减少目标伤害 |
| 冰霜之心 | 减少附近敌人攻速 |
| 兰顿之兆 | 减少 20% 暴击伤害 |
| 死亡之舞 | 30% 伤害转化为流血 |
| 石像鬼石板甲 | 主动获得巨大护盾 |
| 钢铁烈阳之匣 | 提供护盾 |
| 振奋盔甲 | 增强护盾和治疗效果 |
| 自然之力 | 叠层获得魔抗和移速 |
| 兰顿之兆主动 | 减少范围内敌人 10% AD 伤害 |

### 12.2 伤害加深

| 来源 | 效果 |
|------|------|
| 黑暗收割 | 对低血量英雄造成额外伤害 |
| 电刑 | 独立伤害触发 |
| 征服者 | 叠满后提供自适应力和全能吸血 |
| 致命节奏 | 提升攻速上限 |

### 12.3 最终伤害公式 (完整版)

```
最终伤害 = 原始伤害 × 抗性乘数 × 伤害加深 × (1 - 减伤%) × 护盾吸收
```
