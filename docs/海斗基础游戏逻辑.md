# 海斗（极地大乱斗 / 海克斯大乱斗）基础游戏逻辑

> 数据来源：LOL 嚎哭深渊 / 海克斯大乱斗官方机制 + 玩家社区攻略整理
> 最后更新：26.12

---

## 一、出生与复活

### 1.1 出生规则
- **初始等级 = 3 级**，自带 3 个技能点可立即分配
- **初始金钱 = 1375-1400**（海斗 2.0+ 通行证 4 级玩家 +300 = 1675-1700）
- 出生位置 = **己方泉水 / 水晶枢纽**
- 海斗 2.0 通行证 18 级玩家 **开局自带 1 个属性锻造器**
- 出生时双方 5 名英雄立即就位，无选路环节

### 1.2 死亡 / 复活机制
- 死亡后自动回泉水，**等复活时长由等级决定**（约 10-30 秒）
- 海斗 3.0 模式（斗魂竞技场）：濒死状态 11-8 秒（基于回合），友军站在身旁 3 秒可复活
- 复活时获得短暂无敌帧 + 移动速度 + 韧性加成
- 复活后 1.5 秒内**免受伤害 + 不可被选取**，并获得衰减的 30% 伤害减免

## 二、泉水可做的事情

在己方泉水位置可以：

### 2.1 购买 / 出售装备
- 泉水内装备商店全部开放（部分道具**禁止购买**，见下）
- 海斗专属装备：守护者号角 / 战锤 / 法球 / 之刃（**传说级**），可蹭神话被动
- 装备带来的**血量 / 法力直接增加**，但**吸血技能回复不会在泉水恢复**

### 2.2 选择海克斯（如果有未选择的）
- 升级到 3/7/11/15 级时，若有未选海克斯，会弹出选择界面
- 死亡时也会触发（如果回合数达到选择节点）
- 每局最多选 **4-5 个**海克斯（取决于版本）

### 2.3 坐海克斯飞门（传送门）到己方外塔
- 泉水水晶前有**海克斯飞门印记**
- 冷却时间 **30 秒**
- 落点**固定在己方一塔（外塔）范围内**
- 一塔被摧毁后，**终点印记自动变更到二塔**（内塔）
- 二塔被毁则继续后移到水晶前

> **重要**：玩家不能像召唤师峡谷海克斯飞门那样自由选择落点位置

### 2.4 泉水不治疗（核心机制）
- **泉水不能恢复生命值和法力值** ← 海斗最大特色
- 只有**装备/符文/技能**提供的属性会直接增加
- **召唤师技能"回城"禁用**：按 B 仍可触发回城动画与音效，但没有实际效果
- 玩家不能"在泉水挂机"——必须出泉水参与战斗

## 三、地图机制

### 3.1 地图结构
嚎哭深渊为**单线地图**，与召唤师峡谷 3 线不同：
- 双方各有 1 条**兵线**（河道）
- 地图结构（从蓝方基地 → 红方基地）：
  ```
  泉水 → 门牙塔 → 水晶 → 内塔后草丛 → 内塔 → 外塔后草丛 → 外塔 → 河道草丛①② → 敌方外塔 → 敌方内塔 → ... → 敌方水晶
  ```
- 总共 6 个塔（双方各 3 个：外塔/内塔/门牙）+ 2 个水晶 + 4 个草丛

### 3.2 道路医疗包
- 道路上**随机刷血包**，4 个位置（双方外塔两侧）
- 拾取后**2.5 秒**产生治疗光环
- 拾取者立即回 8% 已损生命值和法力值
- 第二段光圈命中回 16% 已损生命值和法力值
- **两段全吃** ≈ 22.72% 已损回复（坦克/脆皮都合适）
- 琪亚娜吃血包可获得水元素，龙女吃第二段可叠被动

### 3.3 草丛（视野/战术）
- 4 个草丛可藏身，敌人没有视野
- 部分英雄（凯隐、薇等）有草丛专属加成
- 蹲草是海斗核心战术之一

### 3.4 防御塔
- **外塔 / 内塔 / 门牙塔** 三种类型，从外到内
- 防御塔有**护甲 / 攻击范围 / 减伤 / 攻速冷却**
- 英雄对防御塔伤害随时间增长 **0% → 25%**（基于 18:00-30:00 区间）
- **塔被摧毁后，所属方的水晶获得"超级兵 + 强化塔"** 效果
- 塔会优先攻击小兵 → 英雄（有仇恨规则）

### 3.5 水晶枢纽（基地）
- 血量 5000 左右
- 摧毁水晶 = 胜利
- 水晶本身会攻击附近敌人

## 四、兵线机制

### 4.1 兵线刷新
- **第一波小兵 45 秒诞生**（开局）
- 18:00 时：25 秒一波
- 30:00 时：13 秒一波（最密）
- 兵线移速 325-425（随时间增长）

### 4.2 兵种
- **近战兵 / 远程兵 / 炮车**（攻城兵）
- 炮车从**第 3 波开始**，每 3 波 1 个
- 炮车和超级兵**附带暗影阔剑被动**，范围 1100 码

### 4.3 群伤对小兵
- **AOE 技能对小兵只生效 75%** ← 鼓励单体击杀 / 推线

### 4.4 击杀奖励
- 击杀英雄金币奖励 **160**（基础）
- 90 秒内的助攻只获 50% 金币
- 210 秒后才获全额 90 助攻金币

## 五、英雄成长

### 5.1 经验获取
- 击杀英雄、小兵、推塔、存活时间都获得经验
- **每秒被动获得 5 经验**（嚎哭深渊光环）
- 等到 3/7/11/15 级时**强制选海克斯**

### 5.2 等级上限
- 普通模式：**18 级满级**
- 18 级后不再升级，但能继续获得属性

### 5.3 召唤师技能急速
- 海斗地图提供 **70 召唤师技能急速**（约 41.18% CD 缩减）
- 适用于：治疗、闪现、点燃、雪球等

### 5.4 嚎哭深渊光环（地图天赋）
| 效果 | 数值 |
|---|---|
| 召唤师技能急速 | 70 |
| 法力回复 | 每秒 0.15% 最大法力 |
| 经验获取 | 每秒 5 经验 |
| 友方治疗效率 | 50%（时光大招、豹女奶自己等） |
| 附近小兵金币 | +6 金币/小兵 |
| 近战英雄魔抗 | +10 魔抗 |
| 群体伤害对小兵 | 75% 效能 |

### 5.5 英雄专属 BUFF
每个英雄有：
- **输出加成**（如 105% 表示 +5% 输出）
- **承伤修正**（如 95% 表示受到 -5% 伤害）
- **特殊 buff**：韧性 +20、急速 +10、治疗 -10% 等

> 例子（13.13 版本）：
> - 亚托克斯：输出 105% / 承伤 95% / 韧性 +20
> - 辛德拉：输出 105%
> - 金克丝：输出 95% / 承伤 105%
> - 提莫 / 吉格斯：急速 -20 / 承伤 120%

## 六、海克斯系统

### 6.1 选择时机
- 玩家在 **3/7/11/15 级升级时**或**阵亡时**触发
- 系统提供 3 个**随机海克斯**供选择
- 选过的海克斯**不会重复出现**（基础规则）

### 6.2 海克斯等级
- **白银** = 基础属性增益（小 buff）
- **黄金** = 战术玩法重构（中 buff）
- **棱彩** = 颠覆性效果（最强，2 个玩家选后移除）

### 6.3 数量
- 1.0/2.0 经典版：每局最多 **4-5 个**
- 2.0 通行证：13 / 31 级可提升高阶符文重随概率

### 6.4 海斗 2.0 通行证系统
| 通行证等级 | 奖励 |
|---|---|
| 4 | 开局 +300 金币 |
| 13 | 提升高阶符文重随概率 |
| 18 | 开局 1 个属性锻造器 |
| 23 | 9/12/16 级免费合剂 |
| 31 | 提升高阶符文重随概率 |

### 6.5 海斗 2.0 套装（羁绊）
- 收集同套装 2-3-4 个海克斯触发**羁绊效果**
- 9 个套装：俯冲炸弹 / 神龙烈焰 / 掷骰狂人 / 下雪天 / 大法师 / 金币雨 / 完全自动化 / 喂呜喂呜 / 叠角龙
- 凑套威力强，但散搭单卡也可用

## 七、回城 / 死亡 / 装备规则

### 7.1 回城（B 键）
- **回城技能禁用**
- 按 B 仍可触发回城动画 + 音效 + 读条
- 没有实际治疗 / 移动效果

### 7.2 死亡后
- 死亡时立即**回泉水**
- 等复活时长（基于等级和当前死亡次数递增）
- 复活后无敌 1.5 秒

### 7.3 装备购买限制
- **离开泉水后，只有死亡才能购买装备** ← 海斗核心规则
- 不能在河道开商店
- 出泉水后买装备的唯一时机 = **死亡后复活期间**

### 7.4 部分道具禁止购买
- 海斗禁用的道具（如多兰系列调整、一些辅助装）
- 海斗专属装备（守护者系列）可作为传说级

## 八、投降机制

| 投票类型 | 时间条件 | 通过要求 |
|---|---|---|
| 队友挂机投降 | 0-3 分钟 | 系统检测 |
| 早期投降 | 8 分钟 | **5 票全同意** |
| 普通投降 | 12 分钟 | **4 票同意** |

> 小技巧：12 分钟是常规阈值，可避免"超前投降"惩罚

## 九、群体 / 单体机制

- **群体伤害（AOE）对小兵 75% 效能** ← 推线略慢，鼓励配合
- **来自英雄单位 1000 码的伤害降低 15%**（对终极技能无效）
- 击杀 1000 码内敌方英雄获得助攻

## 十、战斗 / 战术

### 10.1 基础战斗循环
1. 双方英雄对线
2. 击杀小兵 / 推掉外塔 / 拿海克斯
3. 团战 / 1v1 / 偷塔
4. 推到对方水晶 = 胜利

### 10.2 海斗的策略维度
- **阵容选择**：坦克 + 开团 + 法师 + 射手 + 辅助（中庸）
- **脆皮 POKE 阵**：前期强，后期乏力
- **坦克 / 重装阵**：前期弱，后期无敌
- **推线 / 偷塔**：坦克偷塔流
- **控制链 / 爆发流**：秒杀后排

### 10.3 海斗的"特殊死亡"
- 死后**立即回泉水**，不存在"慢慢复活"概念
- 高等级英雄死后，复活时间 + 死亡计数器双增长
- 友方英雄也可在英雄**身旁 3 秒"救起"濒死队友**（斗魂竞技场模式）
